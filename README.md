# 中国大陆用户购买比特币教程

这是一个详细的步骤指南，帮助中国大陆用户安全地购买比特币。

## 设计风格

本网站采用 **HAB 品牌 B 模块（Bitcoin）设计风格标准**：

- 🎨 **极简现代**：浅色背景，留白充足，信息层次清晰
- 🟠 **品牌配色**：主色深灰黑（#1A1A1A），强调色比特币橙（#FF9900）
- ₿ **品牌标识**：使用比特币官方 Logo
- 💫 **交互设计**：动画简洁流畅（150-250ms），hover 状态使用橙色
- 📐 **布局原则**：单列居中（最大宽度 960px），体现理性克制与未来感

详细设计规范请参阅：[HAB_B_STYLE_GUIDE.md](./HAB_B_STYLE_GUIDE.md)

## 教程内容

本教程涵盖以下主要步骤：

1. 使用苹果手机（安全考虑）
2. 注册或购买美区 Apple ID
3. 下载交易所（欧易或币安）
4. 下载谷歌浏览器
5. 下载钱包软件（imToken）
6. 交易所账号注册和 KYC 认证
7. C2C 交易购买 USDT 或 BTC
8. 在市场使用 USDT 购买比特币
9. 管理比特币（交易所或转入钱包）

# 中国大陆用户购买比特币教程

这是一个详细的步骤指南，帮助中国大陆用户安全地购买比特币。

## 教程内容

本教程涵盖以下主要步骤：

1. 使用苹果手机（安全考虑）
2. 注册或购买美区 Apple ID
3. 下载交易所（欧易或币安）
4. 下载谷歌浏览器
5. 下载钱包软件（imToken）
6. 交易所账号注册和 KYC 认证
7. C2C 交易购买 USDT 或 BTC
8. 在市场使用 USDT 购买比特币
9. 管理比特币（交易所或转入钱包）

## 本地开发

```bash
npm run dev
```

然后在浏览器访问 `http://localhost:8000`

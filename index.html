安全购买比特币，从这里开始

本教程详细介绍中国大陆用户购买比特币的完整流程。我们强调安全性、合规性和风险认知。

重要提示： 购买加密货币存在风险。请充分了解市场风险，只投入可以承受损失的资金。

使用苹果手机 - 安全第一

为什么选择苹果？

  • ✓ iOS 有更严格的安全审查机制
  • ✓ App Store 对应用的审核更严格
  • ✓ 系统级别的隐私保护更好
  • ✓ 恶意软件风险相对较低
建议： 建议购买全新的苹果手机，专门用于加密货币交易和钱包管理。

注册或购买美区 Apple ID

为什么需要美区 ID？

  • ✓ 中国区 App Store 限制加密货币应用
  • ✓ 美区 App Store 有完整的交易所和钱包应用

方法一：自行注册

  1. 打开 App Store
  2. 点击头像 → 退出登录
  3. 点击"创建新的 Apple ID"
  4. 选择国家/地区为"United States"
  5. 填写邮箱、密码等信息
  6. 支付方式选择"None"（无需付款方式）
  7. 完成邮箱验证

方法二：购买现成的

  • 在一些技术论坛购买已注册的美区 ID
  • 建议购买后立即修改密码
  • 添加你的邮箱作为恢复联系方式
⚠️ 警告： 不要与他人共享 Apple ID，建议为其启用双重认证。

下载钱包软件 - imToken

什么是钱包？

钱包是存储和管理加密资产的工具。imToken 是一个多链钱包，支持 Ethereum、Bitcoin、Solana 等多个区块链。

imToken 的优势

  • ✓ 自托管钱包（你完全掌控资产）
  • ✓ 支持多链资产管理
  • ✓ 内置 DeFi 功能
  • ✓ 中文界面，操作简单
  • ✓ 安全性有保障

下载和创建钱包

  1. 在 App Store 搜索 "imToken"
  2. 安装应用
  3. 打开后选择"创建钱包"
  4. 设置强密码（8位以上）
  5. 备份助记词（12个单词，务必妥善保管）
  6. 在空白处验证助记词顺序
  7. 创建完成
⚠️ 关键提示：
  • ❌ 不要将助记词发给任何人
  • ❌ 不要发到互联网上
  • ❌ 不要截屏保存
  • ✓ 建议纸质备份，保存在安全地方
6

交易所账号注册与 KYC 认证

KYC 是什么？

KYC (Know Your Customer) 是身份验证程序，用于符合反洗钱要求。你需要提交身份证件和照片进行验证。

注册步骤

  1. 打开欧易或币安 App
  2. 点击"注册"或"Sign Up"
  3. 输入邮箱或手机号
  4. 设置强密码
  5. 验证邮箱/短信码
  6. 账户创建完成

KYC 认证步骤

  1. 进入"个人中心" → "身份认证"
  2. 选择身份类型（中国大陆选择"身份证"）
  3. 输入姓名和身份证号
  4. 上传身份证正面照
  5. 上传身份证反面照
  6. 进行人脸识别验证
  7. 等待审核（通常 1-30 分钟）
建议：
  • 照片清晰，四个角都能看到
  • 人脸识别时确保光线充足
  • 避免戴太厚重的眼镜或帽子

安全建议

  • ✓ 启用双因素认证 (2FA)
  • ✓ 选择更强的密码，避免常见词汇
  • ✓ 定期检查登录历史
  • ✓ 不要在公共网络连接上进行大额交易
7

C2C 交易 - 购买 USDT 或 BTC

什么是 C2C 交易？

C2C (Consumer to Consumer) 是用户直接交易，你可以用人民币直接向卖家购买 USDT 或 BTC，而不是通过交易所直接买。

为什么选择 USDT？

  • ✓ 价格稳定（1 USDT ≈ 1 美元）
  • ✓ 溢价率低（价格通常接近官方汇率）
  • ✓ 交易对多（USDT 可以兑换任何币种）
  • ✓ 流动性强（容易买入和卖出）

C2C 购买步骤

  1. 进入交易所 → "C2C 交易"或"快速买币"
  2. 选择买入 USDT 或 BTC
  3. 输入购买数量或金额
  4. 点击"购买"查看可用商家列表

第 7.1 步 - 选择卖家（关键！）

选择商家的标准：

  • ✓ 成交量高：日均成交额 10 万以上
  • ✓ 注册时间长：至少 1 年以上账龄
  • ✓ 认证完善：显示"正商户"或"金牌商户"标签
  • ✓ 好评率高：99% 或以上
  • ✓ 最近活跃：最近几天内有交易记录
⚠️ 避免：
  • ❌ 新账户商家
  • ❌ 好评率低于 98%
  • ❌ 长期不活跃的商家
  • ❌ 成交量极低的商家

支付方式

通常支持：

  • 支付宝 (Alipay)
  • 微信支付 (WeChat Pay)
  • 银行卡转账
建议： 第一次交易选择较小金额（如 500-1000 元）测试，确认流程顺利后再增加金额。
7.2

C2C 购买详细流程

成功交易的完整流程

  1. 确认商家信息
    • 查看商家的成交量、好评率
    • 看商家的历史记录和评价
  2. 点击购买
    • 输入购买数量
    • 确认价格和总额
    • 点击"购买"
  3. 等待商家发起交易
    • 商家确认后会显示支付账户信息
    • 账户信息通常是支付宝、微信或银行卡
  4. 进行支付
    • 使用指定的支付方式转账
    • 金额必须完全一致
    • 转账备注（如果有要求）按商家说明填写
  5. 标记已支付
    • 转账后在交易页面点击"我已转账"或"Transferred"
    • 上传转账凭证截图
  6. 等待商家放币
    • 商家核实收款后会将 USDT/BTC 转到你的交易所钱包
    • 通常 5-30 分钟内完成
  7. 确认收币
    • 确认已收到 USDT/BTC 后点击"确认收币"
    • 交易完成
交易安全提示：
  • ✓ 始终使用交易所提供的交易界面，不要私下转账
  • ✓ 不要透露你的真实身份信息给卖家
  • ✓ 保留所有交易凭证截图
  • ✓ 有疑问时及时联系交易所客服
8

在市场购买比特币

现在你有 USDT 了！

接下来可以在交易所的币币交易市场使用 USDT 购买 BTC。

购买方式

方式 1：直接买入（最简单）

  1. 进入交易所 → "币币交易"
  2. 搜索 BTC/USDT 交易对
  3. 输入购买数量或总额
  4. 确认价格
  5. 点击"买入"
  6. 订单立即成交

方式 2：挂单（更划算，但需要等待）

  1. 进入交易所 → "币币交易"
  2. 搜索 BTC/USDT 交易对
  3. 切换到"限价单"或"Limit Order"
  4. 设置你想要购买的价格（通常低于当前市价 0.5%-1%）
  5. 输入购买数量
  6. 点击"买入"
  7. 等待订单成交（可能需要几分钟到几小时）

两种方式对比

特性 市价买入（Market Order） 挂单买入（Limit Order）
成交速度 立即成交 需要等待
价格保证 当前市价成交 自己设定的价格
费用 Taker 手续费（通常 0.1%） Maker 手续费（通常 0.1%）
风险 买到你想要的价格 可能成交不了
建议策略：
  • 小额购买：直接市价买入，速度快
  • 大额购买：可以分几次挂单，能获得更优价格
  • 分批建仓：分多次购买，分散风险
9

管理你的比特币

购买比特币后的选择

购买完成后，你需要决定如何存管你的比特币。主要有两种选择：

选项 1：保留在交易所

优点：
  • ✓ 随时可以快速卖出
  • ✓ 操作简单
  • ✓ 交易所通常有保险
  • ✓ 可以参与交易所生息
缺点：
  • ✗ 交易所存在被黑客攻击风险
  • ✗ 交易所倒闭会造成损失
  • ✗ 不是真正属于你（你没有私钥）

选项 2：提币到自己的钱包

优点：
  • ✓ 完全掌控你的资产
  • ✓ 最高的安全性
  • ✓ 不受交易所影响
  • ✓ 拥有私钥，真正属于你
缺点：
  • ✗ 提币有网络费用
  • ✗ 提币需要等待确认（通常 10-30 分钟）
  • ✗ 卖出需要先转回交易所

提币到 imToken 钱包的步骤

  1. 打开 imToken 钱包
  2. 选择 Bitcoin 钱包
  3. 点击"接收"获取你的 BTC 地址
  4. 复制地址（或生成二维码）
  5. 在交易所进入"提币"或"Withdraw"
  6. 选择提币币种：Bitcoin (BTC)
  7. 粘贴你的 BTC 地址
  8. 输入提币数量
  9. 确认网络费用
  10. 点击"提币"
  11. 可能需要二次验证（邮箱或短信）
  12. 等待交易所处理（通常 1-30 分钟）
  13. 等待区块链确认（通常 30 分钟内）
  14. 在 imToken 中查看到账的 BTC
⚠️ 提币要点：
  • ✓ 第一次提币建议先转小额测试
  • ✓ 务必确认地址正确（检查多次）
  • ❌ 不要发送到错误的地址（无法恢复）
  • ✓ 保存好收据和交易 hash

建议的存储方案

  • 短期交易： 保留在交易所，便于快速买卖
  • 中期持有： 一半在交易所，一半在钱包
  • 长期保存： 转入自托管钱包（imToken 或硬件钱包）
  • 大额资金： 使用硬件钱包（Ledger、Trezor 等）

❓ 常见问题解答

购买比特币本身在中国大陆是合法的。你可以拥有加密货币。但是：

  • 中国禁止加密货币交易所在国内运营
  • 你可以通过国外交易所购买
  • 购买和持有比特币不违法
  • 大额兑换人民币时要合规

建议咨询专业人士了解最新政策。

没有最低要求。你可以购买任何金额：

  • 可以购买部分比特币（BTC 最小单位是 1 satoshi = 0.00000001 BTC）
  • 从几十块到几千块都可以
  • 交易所通常有最小订单限制（如 10 USDT）

建议新手从小额开始，逐渐增加。

C2C 交易在正规交易所上是相对安全的：

  • ✓ 交易所充当中介，保护买卖双方
  • ✓ 钱必须先到交易所托管
  • ✓ 成交后才能确认收币
  • ✓ 有纠纷可以申请客服仲裁

关键是选择信誉好的商家（高成交量、高好评率、账龄长）。

如果你丢失了助记词：

  • ❌ 无法恢复（没有中央服务器可以重置）
  • ❌ 钱包内的资金无法访问
  • ⚠️ 这是自托管钱包的代价和风险

预防措施：

  • ✓ 创建钱包时立即备份助记词
  • ✓ 使用至少 2 份纸质备份
  • ✓ 分别存放在两个安全地点
  • ✓ 不要存在电脑或手机上

时间：

  • 交易所处理：通常 1-30 分钟
  • 区块链确认：通常 10-30 分钟（BTC）
  • 总时间：通常 30-60 分钟

手续费：

  • 交易所提币费：通常 0.0005-0.001 BTC
  • 网络费用：根据拥堵情况变化
  • 总费用：通常 5-100 元人民币

是的，比特币价格波动很大。这是投资风险：

  • 📈 可能上涨 10-50% 或更多
  • 📉 也可能下跌 10-50% 或更多
  • ⚠️ 这种波动性是加密货币的特点

降低风险的方法：

  • ✓ 只投入可以承受损失的资金
  • ✓ 分批购买（平均成本法）
  • ✓ 长期持有（通常 1 年以上）
  • ✓ 不要频繁交易

有很多其他自托管钱包选择：

  • 移动钱包：
    • - Trust Wallet
    • - MetaMask
    • - imToken（推荐）
  • 硬件钱包：
    • - Ledger Nano S/X（推荐）
    • - Trezor

对于初学者，imToken 已经很好用了。等资产增加后可以考虑硬件钱包。

这就是为什么自托管很重要：

  • ✓ 如果你把币提到自己钱包，交易所倒闭不影响你
  • ✗ 如果币还在交易所，可能会损失

选择可信交易所：

  • ✓ 选择老牌、大型交易所（币安、欧易等）
  • ✓ 它们通常有巨额保险
  • ✓ 长期运营经验

建议：大额资金不要长期放在交易所，要转到自托管钱包。
